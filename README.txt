Proyecto Flutter Web - Invitación de Boda (ejemplo)
Contenido:
- lib/main.dart : código principal (pantalla sobre -> invitación)
- assets/images/* : imágenes de ejemplo (placeholders)
- assets/audio/background.wav : archivo de audio silencioso de ejemplo

Cómo usar:
1. Coloca el ZIP en tu máquina con Flutter instalado.
2. Descomprime.
3. Desde la carpeta del proyecto ejecuta:
   flutter pub get
   flutter run -d chrome

Notas:
- Reemplaza assets/images/*.png por tus fotos reales.
- Reemplaza assets/audio/background.wav por la pista instrumental que prefieras.
- Cambia el número de WhatsApp en lib/main.dart (variable whatsappTo) por tu número (ej: 573000000000).
- El contador de confirmados es local (se reinicia al recargar).
